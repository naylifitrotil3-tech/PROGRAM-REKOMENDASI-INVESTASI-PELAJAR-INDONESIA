# PROGRAM-REKOMENDASI-INVESTASI-PELAJAR-INDONESIA
Aplikasi berbasis Python ini membantu teman teman pelajar menemukan jenis investasi yang cocok dengan kondisi mereka dimulai dari rincian pengelolaan uang bulanan, hasil analisis dari kebiasaan pengelolaan uang, sampai rekomendasi saham, serta keuntungan dan kerugian.
